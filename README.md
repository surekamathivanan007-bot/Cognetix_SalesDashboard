# Interactive Sales Dashboard

Project 6 - Intermediate Stage
Internship: Cognetix Technology

## About
Built an interactive sales dashboard using Power BI to display KPIs such as total sales, profit, orders, category performance and regional breakdown.

## Dataset
Download from Kaggle: https://www.kaggle.com/datasets/juhi1994/superstore
Place the file as US Superstore data.csv in the project folder.

## Tools used
- Power BI Desktop
- DAX
- Power Query Editor

## What the dashboard includes
- KPI cards (Sales, Profit, Quantity)
- Category wise sales bar chart
- Region wise sales pie chart
- Sales over time line chart
- Region slicer for interactive filtering

## How to open
1. Download Power BI Desktop from https://powerbi.microsoft.com/desktop/
2. Open the Cognetix_SalesDashboard.pbix file
